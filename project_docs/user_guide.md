# User Guide

This guide explains how to use the Multi-Context RAG Chatbot.

---

## How to use the chatbot

1. Open the application
2. Type your question in the input box
3. Press Enter
4. The chatbot will respond with an answer

---

## Types of questions you can ask

### General Questions
- What is machine learning?
- Explain AI in simple terms

### NEC (Electrical Code)
- What does NEC say about grounding?
- Explain Article 250

### Wattmonk
- What does Wattmonk do?
- What services does Wattmonk provide?

---

## Follow-up questions

You can ask follow-up questions like:

Example:
- What does NEC say about grounding?
- Explain more

The chatbot will try to understand the previous context.

---

## Sidebar features

- Clear Chat → resets the conversation
- Sample queries → quick examples to try

---

## Notes

- If a question is too vague, the chatbot may ask you to rephrase
- For best results, ask clear and specific questions
